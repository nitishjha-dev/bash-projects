# System Monitoring Dashboard

## Description

This script shows basic system information like CPU usage, memory usage, disk usage, and uptime. I made this to understand how system monitoring works in Linux.

## Features

* Shows CPU usage
* Displays memory usage
* Shows disk usage
* Displays system uptime
* Gives alert if usage is high

## How to Run

```bash
chmod +x System_Monitoring_Dashboard.sh
./System_Monitoring_Dashboard.sh
```

## Example

When you run the script, it prints system stats in the terminal with usage percentages.

## Use Case

Useful for monitoring system health. This kind of script is used in servers and DevOps environments.
