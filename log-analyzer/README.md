# Log Analyzer

## Description

This Bash script analyzes a log file and extracts useful information like errors, failed login attempts, and IP addresses. I built this to understand how logs are handled in Linux systems.

## Features

* Counts total lines in log file
* Detects error messages
* Counts failed login attempts
* Extracts unique IP addresses

## How to Run

```bash
chmod +x Log_Analyzer.sh
./Log_Analyzer.sh
```

## Example

You can use the sample.log file provided and run the script to see results.

## Use Case

This script can be useful in debugging and basic security analysis. It is similar to how logs are analyzed in real systems.
